Accesibilidad en Coordicanarias

Información sobre las pautas WCAG 2.2 AA de accesibilidad implementadas en nuestra web

Nuestro compromiso con la accesibilidad

Coordicanarias se compromete a garantizar que su sitio web sea accesible para todas las personas, independientemente de sus capacidades o del dispositivo que utilicen para acceder a la información. Esta declaración de accesibilidad se aplica al sitio web https://coordicanarias.com y refleja nuestro compromiso con la inclusión digital y el acceso universal a la información.

Fecha de publicación: 24 de noviembre de 2025

Última revisión: 24 de noviembre de 2025

Esta declaración fue preparada y publicada en esta fecha. Se actualizará cuando se realicen cambios significativos en el sitio web o cuando se detecten nuevos problemas de accesibilidad.

Coordicanarias se compromete a mantener y mejorar continuamente la accesibilidad de este sitio web. Realizamos revisiones periódicas y actualizaciones para garantizar que cumplimos con los más altos estándares de accesibilidad. Agradecemos cualquier sugerencia o comentario que nos ayude a mejorar la accesibilidad de nuestro sitio web y, por extensión, el acceso a la información sobre nuestros servicios para todas las personas.

Accesibilidad en Coordicanarias

AA

WCAG 2.2 Nivel AA

Nivel de accesibilidad

Este sitio web cumple con el Nivel AA de las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.2 del World Wide Web Consortium (W3C).

El nivel AA representa el más alto grado de accesibilidad posible, garantizando que el contenido sea accesible al mayor número de personas, incluyendo aquellas con diversidad funcional visual, auditiva, motora y cognitiva.

Criterios de conformidad implementados

  • Ratios de contraste máximos (21:1 cuando es posible)
  • Navegación completa mediante teclado
  • Compatibilidad total con lectores de pantalla
  • Textos alternativos descriptivos para todas las imágenes
  • Estructura semántica correcta del código HTML
  • Opciones de personalización de fuentes y colores

Estándar WCAG 2.2

Las WCAG 2.2 son el estándar internacional más reconocido para la accesibilidad web, desarrolladas por el W3C.

Consultar WCAG 2.2 →

Tecnologías utilizadas

  • HTML5 semántico
  • CSS3 con diseño responsive
  • JavaScript para funcionalidades mejoradas
  • ARIA (Accessible Rich Internet Applications) para elementos interactivos

Metodología de evaluación

Este sitio web ha sido evaluado mediante autoevaluación, metodología reconocida por el artículo 17.2 del Real Decreto 1112/2018, que permite a organizaciones sin ánimo de lucro y entidades del tercer sector realizar evaluaciones internas siguiendo la metodología WCAG-EM (Website Accessibility Conformance Evaluation Methodology) del W3C.

Base legal: Artículo 17.2

El artículo 17.2 del Real Decreto 1112/2018 establece que la autoevaluación es una metodología válida para verificar el cumplimiento de los requisitos de accesibilidad, especialmente para organizaciones sin ánimo de lucro y del tercer sector.

Esta disposición reconoce que las entidades del tercer sector pueden realizar evaluaciones internas rigurosas siguiendo metodologías estandarizadas como WCAG-EM.

Metodología WCAG-EM

WCAG-EM (Website Accessibility Conformance Evaluation Methodology) es la metodología oficial del W3C para evaluar la conformidad con WCAG.

Esta metodología proporciona un enfoque estructurado y reproducible para determinar el grado de conformidad de un sitio web con las Pautas WCAG.

Herramientas utilizadas

  • Validador de HTML del W3C
  • Validador de CSS del W3C
  • WAVE (Web Accessibility Evaluation Tool)
  • axe DevTools
  • Lighthouse (Google Chrome)
  • Navegadores con lectores de pantalla (NVDA, JAWS)
  • Pruebas de navegación por teclado
  • Análisis de contraste de color

Aspectos evaluados

  • Perceptibilidad del contenido
  • Operabilidad de la interfaz
  • Comprensibilidad de la información
  • Robustez del código
  • Compatibilidad con tecnologías asistivas
  • Ratios de contraste de color
  • Navegación por teclado
  • Estructura semántica del HTML

Marco normativo

Esta declaración de accesibilidad se fundamenta en el siguiente marco normativo español y europeo:

Real Decreto 1112/2018

Sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público.

Artículo 17.2: Permite la autoevaluación como metodología válida para verificar el cumplimiento de los requisitos de accesibilidad, especialmente para entidades sin ánimo de lucro y organizaciones del tercer sector.

Directiva UE 2016/2102

Del Parlamento Europeo y del Consejo sobre la accesibilidad de los sitios web y aplicaciones móviles de los organismos del sector público.

Establece los estándares europeos de accesibilidad digital y promueve la adopción de las Pautas de Accesibilidad al Contenido Web (WCAG).

Otras normativas aplicables

  • Norma UNE-EN 301549:2022 sobre requisitos de accesibilidad para productos y servicios TIC
  • Ley 11/2007, de 22 de junio, de acceso electrónico de los ciudadanos a los Servicios Públicos

Características de accesibilidad implementadas

Este sitio web incorpora las siguientes características para garantizar la máxima accesibilidad:

Navegación y estructura

  • Menús de salto al contenido
  • Estructura HTML5 semántica
  • Navegación completa por teclado
  • Orden lógico de tabulación

  • Ajuste de tamaño de fuente
  • Modos de alto contraste

Compatibilidad técnica

  • Etiquetas ARIA descriptivas
  • Compatible con lectores de pantalla
  • Responsive design
  • Control de animaciones
  • Formularios accesibles

Comunicar problemas de accesibilidad

Si encuentra alguna barrera de Accesibilidad en este sitio web o necesita información en un formato alternativo, por favor póngase en contacto con nosotros. Agradeceríamos una descripción detallada del problema encontrado, indicando la página o sección específica del sitio web, el navegador, el sistema operativo que usas. Responderemos a su solicitud en un plazo máximo de 10 días hábiles.
