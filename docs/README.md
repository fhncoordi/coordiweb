## Description

**Labinator A11y-Bootstrap Template** is a perfectly accessible template according to the WCAG 2.2 Level AAA Guidelines. It is also compatible with Section 508 and the ADA guidelines. All of its codes are valid according to the W3C and optimized perfectly for speed. It is also fully responsive and amazingly elegant suitable for any niche, app, website, or project. It highly customizable, easy to use, and comes with an advanced fully-functional accessibility toolbar.

## Main Features

- Valid Modern Code.
- Lightweight & Fast.
- Fully Responsive.
- Accessibility Ready (WCAG Level AAA).
