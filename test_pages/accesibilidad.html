Declaración de Accesibilidad

Compromiso con la accesibilidad universal

Nuestro compromiso con la accesibilidad

Coordicanarias se compromete a garantizar que su sitio web sea accesible para todas las personas, independientemente de sus capacidades o tecnologías de acceso. Trabajamos continuamente para mejorar la experiencia de usuario y aplicar los estándares de accesibilidad pertinentes.

Nivel de conformidad

Este sitio web cumple con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.2 de nivel AAA. Estas pautas explican cómo hacer que el contenido web sea más accesible para personas con discapacidad.

El cumplimiento con el nivel AAA significa que hemos implementado los criterios de conformidad más exigentes, incluyendo:

  • Ratios de contraste máximos (21:1 cuando es posible)
  • Navegación completa mediante teclado
  • Compatibilidad total con lectores de pantalla
  • Textos alternativos descriptivos para todas las imágenes
  • Estructura semántica correcta del código HTML
  • Opciones de personalización de fuentes y colores

Información sobre esta declaración

Fecha de publicación: 24 de noviembre de 2025

Última revisión: 24 de noviembre de 2025

Tecnologías utilizadas:

  • HTML5 semántico
  • CSS3 con diseño responsive
  • JavaScript para funcionalidades mejoradas
  • ARIA (Accessible Rich Internet Applications) para elementos interactivos

Metodología de evaluación

La accesibilidad de este sitio web ha sido evaluada mediante autoevaluación, metodología reconocida por el artículo 17.2 del Real Decreto 1112/2018, que permite a organizaciones sin ánimo de lucro y entidades del tercer sector realizar evaluaciones internas siguiendo la metodología WCAG-EM (Website Accessibility Conformance Evaluation Methodology) del W3C.

Autoevaluación

Evaluación interna siguiendo la metodología WCAG-EM (Website Accessibility Conformance Evaluation Methodology) para garantizar el cumplimiento de todos los criterios de éxito.

Base legal: Artículo 17.2 del Real Decreto 1112/2018, que valida la autoevaluación como metodología para verificar el cumplimiento de requisitos de accesibilidad.

Herramientas automáticas

Validadores de HTML, validadores de contraste de color, herramientas de análisis de accesibilidad automáticas y extensiones de navegador especializadas.

Pruebas manuales

Navegación con teclado, pruebas con lectores de pantalla (NVDA, JAWS), verificación de navegadores y dispositivos diversos, y validación de formularios y contenido interactivo.

Pruebas con usuarios

Colaboración con usuarios con diferentes tipos de discapacidad para validar la experiencia real de uso del sitio web.

Características de accesibilidad implementadas

  • Menú de accesibilidad: Panel de configuración con opciones de fuente, tamaño, contraste y saturación
  • Menú de salto: Enlaces para saltar directamente a las secciones principales del contenido
  • Estructura clara: Uso correcto de encabezados jerárquicos (H1-H6)
  • Textos alternativos: Todas las imágenes incluyen descripciones alternativas
  • Contraste óptimo: Ratios de contraste que cumplen o superan los 21:1
  • Navegación por teclado: Todas las funcionalidades accesibles sin ratón
  • Compatibilidad con lectores de pantalla: Etiquetas ARIA y roles semánticos
  • Diseño responsive: Adaptación a diferentes tamaños de pantalla y dispositivos
  • Formularios accesibles: Etiquetas claras y mensajes de error descriptivos
  • Enlaces descriptivos: Textos de enlace que describen claramente su destino

Limitaciones conocidas

Aunque nos esforzamos por mantener los más altos estándares de accesibilidad, algunas áreas pueden presentar limitaciones:

Documentos PDF externos: Algunos documentos descargables pueden no cumplir completamente con los estándares de accesibilidad. Estamos trabajando para actualizar y etiquetar correctamente todos los documentos.

Contenido multimedia de terceros: Algunos videos o contenidos embebidos de plataformas externas pueden no incluir subtítulos o audiodescripciones.

Si encuentra dificultades para acceder a cualquier contenido, puede solicitar una versión alternativa contactando con nosotros.

Marco normativo

Esta declaración de accesibilidad se elabora en cumplimiento de:

  • Real Decreto 1112/2018, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público. Especialmente el artículo 17.2, que establece que la autoevaluación es una metodología válida para verificar el cumplimiento de los requisitos de accesibilidad, especialmente para organizaciones sin ánimo de lucro y del tercer sector.
  • Norma UNE-EN 301549:2022 sobre requisitos de accesibilidad para productos y servicios TIC
  • Directiva (UE) 2016/2102 del Parlamento Europeo y del Consejo
  • Ley 11/2007, de 22 de junio, de acceso electrónico de los ciudadanos a los Servicios Públicos

Compromiso de mejora continua

En Coordicanarias nos comprometemos a mantener y mejorar continuamente la accesibilidad de nuestro sitio web. Realizamos revisiones periódicas, incorporamos las últimas mejores prácticas y escuchamos activamente el feedback de nuestros usuarios para garantizar que todas las personas puedan acceder a nuestros servicios e información de manera equitativa.

La accesibilidad no es un destino, es un compromiso continuo.
